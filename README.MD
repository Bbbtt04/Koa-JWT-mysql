## 介绍

- 💪 这是一个对于前端人员,入门后端的一个非常好的项目
- 💌 支持用户的增添改查以及 JWT 鉴权,从实战中感受 koa2 的精髓
- 我们将选择 TypeScript 作为开发语言，数据库选用 MySQL，并使用 TypeORM 作为数据库桥接层。
